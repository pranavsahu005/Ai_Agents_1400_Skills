## Repository Guidelines
This repository is an experimental AI lab containing multiple independent projects and research experiments:
- **3d podfolio**: A React-based 3D portfolio using Vite, GSAP, and Framer Motion. Located in 3d podfolio/.
- **prisma-studio**: A React-based studio application focusing on GSAP animations and UI effects. Located in prisma-studio/.
- **Expriments**: A collection of HTML/JS experiments (Liquid, API, Illustrated effects). Located in Expriments/.
- **imgs**: Centralized image assets for the lab.
- **AI-Lab-Skills**: A massive registry of agentic skills and documentation.
