# Trickle AI - Site Map & Navigation Structure

## Core Structure
1. **Welcome/Onboarding Screen**
   - Start with a Template (Opens Template Gallery)
   - Start Blank (Triggers Interactive Tutorial)

2. **Board Management Home Screen (Board Gallery)**
   - Grid view of all saved canvases (thumbnails)
   - Search & Filter bar
   - Actions: Create New, Open, Duplicate, Rename, Delete
   - Folders/Organization sidebar

3. **Infinite Canvas Workspace (Main Application)**
   - **Top Navigation Bar:**
     - Home Button (returns to Board Gallery)
     - Board Title (editable)
     - Mode Switch Toggle (Life Canvas <-> Flow Map)
     - "Go To Section" Dropdown
     - Save Status Indicator ("Last saved: X mins ago")
     - File Menu (Version History, Export options)
   - **Main Content Area:**
     - Virtual Canvas Viewport
     - Smart Nodes & Connectors
     - Minimap (Bottom Right)
   - **Floating Toolbar (Bottom Center):**
     - Changes dynamically based on Active Mode (Life vs Flow)
   - **Slide-out Panels:**
     - Left/Right edges: Node Content Panel (when node double-clicked)
     - Right edge: Unified Media Library Panel
     - Tag Cloud / Search Panel
     - Sticker Library Overlay

4. **Modals & Overlays**
   - Global Search Overlay (Ctrl+F)
   - Export Modal (PNG, PDF, SVG, JSON)
   - Version History Sidebar
   - Settings Modal (Themes, Preferences)
   - Template Gallery Overlay
   - Help / Keyboard Shortcuts Modal
