# Antigravity Hooks

A collection of templates, hooks, and configurations to optimize your workflow with the **Antigravity** agent.

## Contents

*   **Installer (`install.sh`)**: Automatic environment setup script (Python, venv, dependencies).
*   **Context (`.context/`)**: System prompts (`system_prompt.md`) and style guides (`coding_style.md`) to align the agent.
*   **Mission (`mission.md`)**: Base template for defining project objectives.

## Quick Start

Run the installer to set up your environment:

```bash
./install.sh
```

Configure your keys in `.env`, and you are ready to work with Antigravity.
