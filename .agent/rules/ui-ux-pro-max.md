# UI UX Pro Max Rules

This project uses design intelligence from [nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill).

## Design System Principles

1. **Intelligent Design Generation**: Analyze project requirements to generate a tailored design system (patterns, styles, colors, typography) before building.
2. **Modern Aesthetics**: Prioritize Glassmorphism, Minimalism, Bento Grids, and Dark Mode (OLED) for premium feel.
3. **Accessibility (WCAG 2.1 AA)**: Ensure 4.5:1 contrast, visible focus states, and keyboard navigation.

## Style Guidelines

- **Typography**: Use high-quality font pairings (e.g., Inter/Outfit for SaaS, Playfair/Montserrat for Luxury).
- **Animations**: Use smooth transitions (150-300ms). Avoid harsh movements.
- **Interactions**: Add `cursor-pointer` to all clickable elements. Implement hover and active states.

## Anti-Patterns to Avoid

- **No Emojis as Icons**: Use SVGs (Heroicons, Lucide).
- **No Harsh Shadows**: Use soft, layered shadows for depth.
- **No Default Colors**: Avoid browser defaults; use curated HSL/Hex palettes.
- **No Inconsistent Spacing**: Use a strict 4px/8px grid system.
